# ds-gallery
React component to display static images

```javascript
// Import the library
import DsGallery from 'ds-gallery';
// Insert the gallery
<DsGallery
  images={arrayOfImageURLs}
  options={{
    open_new_window: true,
    donload: true,
    animation_duration_ms: null,
    width: null,
    height: null
  }}
/>
```
